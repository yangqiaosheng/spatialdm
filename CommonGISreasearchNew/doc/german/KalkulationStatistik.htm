s^2=(1/(n-1))*Sum(Xi- X )^2

Count value occurrences:

Ein mögliches Szenario der Gruppenentscheidungsfindung nimmt an, daß die Gruppenmitglieder individuell die Menge der Optionen analysieren und anschließend darüber abstimmen. Descartes liefert das Hilfswerkzeug, welches dem Gruppenführer ermöglicht die Ergebnisse der Wahl auszuwerten. Die Stimme jeder Person kann in einer Spalte der Tabelle mit den entsprechenden Optionen gespeichert werden.
Das System gibt dem Gruppenführer die Möglichkeit automatisch die Stimmen auszuzählen (oder in anderen Worten, zählt für jede Reihe der Tabelle die Häufigkeit der vorkommenden Werte aus einer spezifizierten Submenge der Spalten). Die erhaltene Auszählung kann mit sämtlichen zur Verfügung stehenden Darstellungen visualisiert werden.
