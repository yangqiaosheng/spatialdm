Attributwerte anzeigen

Viele graphische Anzeigen die von dem System produziert werden, stellen einige Objekte mit zugehörigen thematischen Daten dar (Werte verschiedener Attribute). Zum Beispiel werden auf einer Karte solche Objekte mit linearen oder räumlichen Symbolen, oder geometrischen Zeichen oder Ikonen dargestellt. In einem "Scatter"-Graph oder einem Punktegraph werden Objekte mittels kleinen Kreisen, in einem Graphen mit parallelen Koordinatenparallel coordinates plot mittels gebrochenen Linien repräsentiert.

Es gibt zwei Wege Attributwerte eines Objektes zu betrachten. Erstens können die Werte in einem popup Fenster angezeigt werden, welches erscheint, wenn der Mauszeiger auf eines graphisches Element zeigt, das ein Objekt repräsentiert (siehe Abb. 1).


Abb. 1
Darstellung eines Attributwertes in einem popup Fensters. Der Mauszeiger deutet auf ein Objekt auf der Karte. Das popup Fenster zeigt die dazugehörigen Werte des Objektes an.

Zweitens kann der Benutzer ein sogenanntes "persistent object view" öffnen (Abb. 2). Wenn der Benutzer ein Objekt aus irgend einer Darstellung auswählt, wird der zugehörige Attributswert dem "persistent object view" hinzugefügt und verbleibt angezeigt, bis der Benutzer das Object explizit wieder abwählt. So ist das "persistent object view" von der Mausbewegung unabhängig und kann die Daten mehrer selektierte Objekte gleichzeitig anzeigen. An- und Abwahl eines Objektes geschieht durch Mausclick oder durch die Einschliessung mit einem rechteckigen Auswahlfensters. Auf der Kartre in Abb. 2 werden markierte Objekte mit einem schwarzen Kreis dargestellt. Ausgewählte Objekte werden in allen graphischen Darstellungen speziell markiert, damit sie von den anderen Objekten besser zu unterscheiden sind.


Abb 2
Darstellung von Attributwerten in einem "persistent object view" integriert in einem Subfenster des Kartenfensters. Momentan selektierte Objekte werden mit einem schwarzen Kreis markiert. Im "object view" werden die zugehörigen Attributwerte angezeigt.

Die Einstellung des popup Fensters oder des "persistent object view" geschieht über die Menüleiste (options->show record options).

Einige wichtige Hinweise zur Darstellung der Attributwerte in einem popup Fenster:

  • Die Darstellung der Attributwerde in einem popup Fenser ist standardmäßig eingeschaltet.
  • Das Fenster wird nur angezeigt, solange die Maus auf das Objekt in einigen Darstellungen zeigt und verschwindet wieder, sobald die Maus bewegt wird.
  • Das Fenster zeigt nur Attributwerte an, die in der Darstellung in der sich die Maus befindet, auch visualiziert werden. So werden zum Beispiel bei der Mausberührung eines Punktes (Kreises) in einem scatter plot, auch nur die zwei dargestellten Attribute angezeigt.
  • In einer Karte werden nur die Daten der Aktiven Schicht angezeigt. Um die Daten anderer Schichten anzeigen zu lassen, muß diese Schicht aktiviert werden.
Wichgite Hinweise zum "persistent object view":
  • "Persistent object view" ist standardmäßig ausgeschaltet. Es kann unter options->show record options eingeschaltet weren.
  • Objekte, die im "object view" dargestellt werden sollen, können von allen Darstellungen aus selektiert werden.
  • An- und Abwählen eines Objektes durch Mausklick (in jeder Darstellung) ist ausgeschaltet, wenn der Mausklick für Kartenmanipulation, wie "visueller Vergleich", benutzt wird. Analog ist das selektieren mit einem Auswahlfenster ausgeschaltet, wenn das Mausziehen zum Vergrößern und Verschieben benutzt wird. Die Interpretation der Mausaktionen verändert man in der "tool bar" des Kartenfensters.
  • Die Auswahl eines Objektes von der Kartes geschieht nur in der aktiven Schicht.
  • Es gibt zwei Modi des "persistent object view": zeige momentan visualizierte Attribute an (in allen Darstellungen die auf dem Bildschirm zu sehen sind) oder zeige nur die vom Benutzer spezifizierten Attribute. Im ersten Modus werden die Attribute vom "object view" automatisch hinzugefügt oder entfernt wenn neue Darstellungen erzeugt oder geschlossen werden. Im zweiten Modus verbleiben bereits selektierte Attribute im "object view" bis sie explizit abgewählt werden. Um zwischen den beiden Modi zu wechseln und um die anzuzeigenden Attribute auszuwählen sollte man das Einstellungsfenster unter options->show record options benutzen.
